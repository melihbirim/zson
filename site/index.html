AI-assisted JSON filtering

Ask questions over JSON records.

zson.ai turns natural language into deterministic zson queries for logs, API payloads, DB JSON rows, event streams, and test fixtures.

View zson on GitHub Download CLI 
$ zson ask "active admins for acme"

query
{ "tenantId": "acme",
  "role": "admin",
  "active": true }

$ zson '{ "role": "admin" }' users.ndjson \
  --assert-count 3

Engine first

zson is the deterministic CLI and Zig library. AI suggests queries; zson executes them.

Fixture ready

Prepare integration-test data and CI assertions without starting a database when records are enough.

Built for records

Filter JSON and NDJSON batches from APIs, queues, logs, exports, and local files.

zson.ai is powered by the open-source zson engine. github.com/melihbirim/zson
